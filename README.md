# GNS3 Lab

GNS3 Home lab: Network configuration with FRR routing, Zabbix monitoring and Grafana dashboards.

## Stack

| Tool    | Purpose                              |
|---------|--------------------------------------|
| GNS3    | Network topology simulation          |
| FRR     | Dynamic routing with OSPF            |
| Zabbix  | Host monitoring and alerting         |
| Grafana | Metrics visualization and dashboards |
